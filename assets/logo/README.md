# Logo Assets

This directory contains logo and icon assets for the namer project.

## Naming Convention

- Rasterized images use the pattern `logo-<size>.png` (e.g., `logo-256.png`, `logo-512.png`)
- Vector images use `logo.svg`
- All filenames are lowercase and hyphen-separated

## Expected Sizes

| File | Dimensions | Purpose |
|------|------------|---------|
| `logo-256.png` | 256x256 | Primary logo used in the project README |
| `logo-512.png` | 512x512 | High-resolution logo for other uses |

`logo-256.png` is the primary logo referenced in the project README.
